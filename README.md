# Syed Abdul Rahim — Portfolio

[![Live Site](https://img.shields.io/badge/Live-sdrahim786.github.io-c2410c?style=flat-square)](https://sdrahim786.github.io)

Personal portfolio site for **Syed Abdul Rahim** — MSc Artificial Intelligence student at Brunel University of London. Building autonomous agents, ML systems, and production software.

## About

A single-page, single-file portfolio built with vanilla HTML, CSS, and JavaScript. No frameworks, no build step, no dependencies beyond two Google Fonts. Deployed via GitHub Pages.

## Stack

- **HTML5 / CSS3** — semantic markup, modern CSS (`color-mix`, custom properties, `clamp()`)
- **Vanilla JavaScript** — custom cursor, live time, scroll-triggered reveals via `IntersectionObserver`, theme persistence
- **Typography** — Geist (sans), Geist Mono, Instrument Serif (display)
- **Hosting** — GitHub Pages

## Features

- Editorial-minimal aesthetic with a warm off-white palette
- Light/dark theme toggle with `localStorage` persistence
- Custom cursor (desktop only, gracefully disabled on touch)
- Live London time in the navigation
- Scroll-triggered section reveals
- Fully responsive down to 320px
- Zero JavaScript framework dependencies

## Run locally

```bash
git clone https://github.com/sdrahim786/sdrahim786.github.io.git
cd sdrahim786.github.io
open index.html
```

Or use a quick local server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy

This repo is configured for GitHub Pages. Pushes to `main` deploy automatically to [sdrahim786.github.io](https://sdrahim786.github.io).
